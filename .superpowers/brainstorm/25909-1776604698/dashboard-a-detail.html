Why this shape

It preserves current dashboard muscle memory. Expansion happens by adding useful side and bottom blocks, not by inventing new navigation or flashy surfaces.

1. Top stays familiar Keep title + date controls. New piece is one plain-English summary card.
2. Middle stays analytical Averages and main chart remain core. This is still dashboard, not journal page.
3. Right rail adds usefulness Put streaks, gaps, strongest changes, quick-add actions in one compact place.
4. Bottom adds expansion Consistency blocks and recent notable entries make issue #63 feel like real growth, not only polish.
